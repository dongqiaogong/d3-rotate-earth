这是一个基于D3.js的旋转地球，可以在地球转动的过程中，显示不同国家的相关金融指标，如美联储基准利率、GDP初值修正、GDP终值、核心CPI、国际贸易收支等。演示视频见根目录下demo.mp4

#### Environment
install Node.js and npm

#### Run
node app.js
open localhost:8888 in browser.
